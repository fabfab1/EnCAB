How to Cite

Citing the project can be done in two ways: through an Open Science Framework DOI (DOI 10.17605/OSF.IO/NMZXT) or through Zenodo DOIs linked to specific versions. The bibliographical entries for the Open Science Framework project in the most common formats are as follows:

APA Buccellati, F. (INSERT DATE ACCESSED HERE). EnCAB. https://doi.org/10.17605/OSF.IO/NMZXT
MLA Buccellati, Federico. “EnCAB.” Open Science Framework, INSERT DATE ACCESSED HERE. Web.
Chicago Buccellati, Federico. INSERT YEAR ACCESSED HERE. “EnCAB.” Open Science Framework. INSERT MONTH AND DAY ACCESSED HERE. doi:10.17605/OSF.IO/NMZXT.

The Zenodo DOIs, on the other hand, should be used only to cite specific versions of the project; this would primarily be useful when comparing two different versions of the EnCAB project.
