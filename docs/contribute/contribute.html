How to Contribute as a User of EnCAB

If you use EnCAB in a publication, please add the bibliographical information to the appropriate page on the GitHub Pages site; if you prefer, you can also send the reference (and, if possible, a PDF of the publication) to the project email address (webmaster.encab[at]gmail{.}com). Note, however, that EnCAB has no permanent curator, so it may be a while before your data is uploaded to the site.

How to Contribute Algorithms

If you have algorithms to contribute, please add them to the GitHub project and update the interactive algorithms by running the Python code. If you prefer, you can also send the algorithm reference (and, if possible, a PDF of the pertinent pages) to the project email address (webmaster.encab[at]gmail{.}com). Note, however, that EnCAB has no permanent curator, so it may be a while before your data is uploaded to the site.

How to Contribute as a Programmer

If you would like to contribute to the programming behind EnCAB (see the programming page for more details) please let us know at the project email address (webmaster.encab[at]gmail{.}com) or contact the project director directly.
