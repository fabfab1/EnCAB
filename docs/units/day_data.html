Unit: Day -- Description

Days are a measurement of time. As it is often unclear if a 24-hour day or a working day is meant, and how long a working day might be, the unit 'Days' should be used only when the literature does not indicate hours. Working days can be as short as 4 hours, for very heavy work, but tend to be 7-8 hours on average.
